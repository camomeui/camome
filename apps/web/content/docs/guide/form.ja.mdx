## 概要

Camome は JavaScript を使用したクライアントサイドでのフォームバリデーション機能は内蔵していません。むしろ、セマンティックでアクセシブルな HTML を簡単に生成することに重点を置いています。

`FormField` コンポーネントを使用することで、 `id` を自動生成して適切な `<label />` と `<input />` を紐付けたり、説明テキスト（エラーも含みます）も付け加えることができます。その際には[`aria-describedby`](https://www.w3.org/TR/wai-aria-1.1/#aria-describedby) 属性によってアクセシブルにマークアップされます。

Camome は可能な限り JavaScript を使わず、HTML ベースの API を提供しているので、[React Hook Form](https://react-hook-form.com/)のような既存のフォームライブラリと簡単に組み合わせることができます。
