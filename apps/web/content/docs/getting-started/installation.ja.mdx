## [Option 1] TypeScript + CSS Modules

### バンドラーの設定

Camome は [CSS Modules](https://github.com/css-modules/css-modules)でスタイリングされています。

### CSS のインポート

CSS のインポートの順番には意味があります。Camome は [CSS Cascade Layers](https://developer.mozilla.org/ja/docs/Web/CSS/@layer)を使用しているため、特に優先順位の定義が最初に行われるように注意してください。

必ず以下のような順番でインポートします:

1. Cascade Layers の優先順位の宣言(`globals.css` etc.)
2. テーマファイル(`@camome/system/style.min.css` etc.)
3. 各コンポーネント(`import { Button } from "@camome/core"` etc.)

例えば Next.js では次のように実装します:

```css:globals.css
@layer cmm, override;

@layer override {
  .Button {
    border-radius: 999px;
  }
}
```

```tsx:_app.tsx
import "@/styles/globals.css"; // 1
import "@camome/system/style.min.css"; // 2
import { Button } from "@camome/core"; // 3

function App() {
  return <Button>Button</Button>;
}

export default App;
```

CSS のインポートが可能なファイルや、順番が保証されるかどうかはバンドラーやフレームワークに依存するので、うまく動かない場合は確認してみてください。
