---
title: "IconButton"
id: "icon-button"
description: "An IconButton is a button that's represented by an icon."
---

## Variants

```tsx live
<>
  <IconButton variant="solid">
    <BellAlertIcon />
  </IconButton>
  <IconButton variant="subtle">
    <BellAlertIcon />
  </IconButton>
  <IconButton variant="outline">
    <BellAlertIcon />
  </IconButton>
  <IconButton variant="ghost">
    <BellAlertIcon />
  </IconButton>
</>;
```
