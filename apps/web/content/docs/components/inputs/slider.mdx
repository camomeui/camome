---
title: "Slider"
id: "slider"
description: "A slider enables the user to input continuous values."
---

## Overview

```tsx live
<div style={{ width: "12rem" }}>
  <Slider aria-label="volume" />
</div>;
```

<Message status="warn">
  Because this component is a bare `input[type="range"]` with some CSS styles,
  colorization of the current value is not supported currently.
</Message>

## Sizes

```tsx live direction="vertical"
<div
  style={{
    display: "flex",
    flexDirection: "column",
    gap: "3rem",
    width: "12rem",
  }}
>
  <Slider size="sm" />
  <Slider size="md" />
  <Slider size="lg" />
</div>;
```
