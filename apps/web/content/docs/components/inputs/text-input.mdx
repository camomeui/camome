---
title: "TextInput"
id: "text-input"
description: "A TextInput lets the user to enter and edit text."
---

## Usage

```tsx live
<FormControl labelText="Name">
  <TextInput required placeholder="John Doe" />
</FormControl>;
```

## States

### Error

```tsx live
<FormControl
  labelText="Name"
  helperText="Tell us your identifier"
  error="Something is wrong"
>
  <TextInput required placeholder="John Doe" />
</FormControl>;
```

### Disabled

```tsx live
<TextInput disabled placeholder="You can't type..." />;
```

## Sizes

```tsx live
<>
  <TextInput size="sm" placeholder="Small" />
  <TextInput size="md" placeholder="Medium" />
  <TextInput size="lg" placeholder="Large" />
</>;
```
