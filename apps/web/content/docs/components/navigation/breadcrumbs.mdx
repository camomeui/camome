---
title: "Breadcrumbs"
id: "breadcrumbs"
description: "A Breadcrumbs trail enables navigation to ascendant routes of the current page."
---

## Usage

```tsx live
<Breadcrumbs
  paths={[
    {
      href: "#home",
      label: "Home",
    },
    {
      href: "#category",
      label: "Category",
    },
    {
      label: "Product",
    },
  ]}
/>;
```
